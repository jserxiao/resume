# 简历 SaaS 平台 — 功能全景

> 本文档全面列举一个简历 SaaS 产品应该具备的功能模块，按业务场景组织。
> 每项标注 P0/P1/P2 优先级，适用于产品路标决策。

---

## 一、用户与账户体系

### 1.1 注册登录

| # | 功能 | 优先级 | 说明 |
|---|------|--------|------|
| 1.1.1 | 邮箱注册 + 密码登录 | P0 | 最基础的账号通道 |
| 1.1.2 | 手机号注册 + 验证码登录 | P0 | 国内用户主流方式 |
| 1.1.3 | 第三方登录（微信 / Google / LinkedIn / GitHub） | P0 | 降低注册门槛 |
| 1.1.4 | 扫码登录（Web 端扫码） | P2 | 企业端场景 |
| 1.1.5 | 单点登录 SSO（SAML / OAuth2） | P1 | 企业订阅需要 |
| 1.1.6 | 密码找回 / 重置 | P0 | 基础必备 |
| 1.1.7 | 多设备登录管理（显示活跃设备、强制登出） | P1 | 安全需求 |

### 1.2 账户管理

| # | 功能 | 优先级 | 说明 |
