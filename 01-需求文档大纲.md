## 一、项目概述

### 1.1 产品背景与定位
- 目标市场：应届生 / 社招求职者 / 高端人才 / HR 与猎头
- 差异化定位：AI 辅助 + 多端协同 + 投递追踪
- 对标产品：WonderCV、木姐、五百丁、Resume.io、FlowCV、Novoresume

### 1.2 核心价值主张
- 一句话说清：用户为什么选择你而不是工具/Word/LaTeX

### 1.3 目标用户画像
| 角色 | 痛点 | 核心需求 |
|------|------|----------|
| 应届生 | 无经验、无模板、无方法 | 手把手引导、AI 润色 |
| 社招跳槽 | 多版本管理、投递记录混乱 | 模板化、一键切换、投递追踪 |
| 高端人才 | 定制化排版、隐私保护 | 深度自定义、定向投递 |
| HR/猎头 | 大量简历筛选、统一格式 | 解析入库、标签筛选 |
| 外包代写 | 批量产出、仿写 | 克隆模板、批量导出 |

### 1.4 商业模式
- Freemium 免费层（3 份以内 / 水印导出）
- 订阅制（月 / 季 / 年，按导出次数分级）
- 增值服务（AI 润色点数 / 代写 / 专业审校）
- 企业端（HR 招聘发布 / 简历解析 / 人才库）
- 联盟分成（和招聘平台合作引流）

## 二、功能需求

### 2.1 用户系统

#### 2.1.1 注册与登录
- 邮箱注册 / 手机号验证（国际区号支持）
- 第三方登录（微信 / 钉钉 / 飞书 / Google / LinkedIn）
- SSO（企业版，OAuth2 / SAML）
- 多设备同时登录、会话管理

#### 2.1.2 账户管理
- 个人信息（姓名、头像、联系方式、求职意向）
- 密码 / 绑定社交账号管理
- 账户注销与数据导出（GDPR 合规）
- 订阅与付费管理（支付方式、发票、退款）

#### 2.1.3 权限分级
| 角色 | 功能范围 |
|------|----------|
| Free 用户 | 编辑、基础模板、3 份简历、水印导出 |
| Pro 用户 | AI 润色、无限制简历、无水印、多格式导出 |
| Enterprise 用户 | 团队管理、HR 筛选台、API 接入 |
| Admin | 平台管理后台 |

### 2.2 编辑器 —— 核心模块

#### 2.2.1 富文本块编辑器
- 所见即所得（WYSIWYG），基于 ProseMirror / Slate.js / TipTap
- 块级拖拽排序（工作经历、教育、项目等区块）
- 内置常用块类型：
  - 基本信息（头像、姓名、联系方式）
  - 工作经历（公司、时间、职位、描述、技术栈标签）
  - 教育背景（学校、学位、GPA、课程列表）
  - 项目经验（名称、链接、角色、描述）
  - 技能标签（带熟练度星级、关键字匹配分）
  - 证书 / 语言 / 出版 / 其他自定义块
- 块可折叠、可隐藏（调整投递侧重点）
- 实时字数统计、建议字数预警

#### 2.2.2 模板系统
- 模板库：按行业（IT/金融/设计/医疗/教育…）、风格（简约/创新/传统/高端）、经验层级分类
- 模板结构：布局（单栏/双栏）、配色、字体、间距、图标
- 用户可基于模板派生自定义变体
- 模板热更新（已创建简历可选择性迁移到新版模板）

#### 2.2.3 多版本管理
- 同一简历派生多个版本（不同公司/岗位）
- 版本对比 diff（V 1.0 → V 2.0 变化高亮）
- 版本命名、克隆、过期提示

#### 2.2.4 富媒体支持
- 头像裁剪、滤镜
- 作品链接嵌入（GitHub / Behance / Dribble / 作品集 PDF）
- 视频简历（嵌入 Vimeo / Bilibili 链接）

### 2.3 AI 功能

#### 2.3.1 智能写作辅助
- JD 解析：粘贴职位描述 → 自动抽取关键词、优先展示相关技能
- 工作描述润色：单句 / 全段 -> 更专业的表达（保留事实、调整动词、量化成果）
- 自动总结：从长段落提取 bullet points
- 经历扩写：从三五个关键字展开成完整描述
- 多语言翻译：中文 → 英/日/德/法（保留排版、专业术语）
- 语法检查与拼写纠错（多语言）

#### 2.3.2 AI 简历评分
- 综合评分（内容完整度 + 关键词匹配度 + 可量化程度 + 排版友好度）
- 每一块给出改进建议（高优/中/低）
- ATS 通过率预估（基于主流 ATS 规则模拟）

#### 2.3.3 智能排版
- 自动调整页边距、字号、行距以达到一页纸
- 智能换行（避免寡妇行/孤儿行）
- 根据内容长度自动推荐模板布局

#### 2.3.4 AI 面试准备
- 从简历内容生成面试可能问题列表
- STAR 法则拆解工作经历
- 模拟面试对话练习

### 2.4 导出与发布

#### 2.4.1 导出格式
| 格式 | 特点 | 付费墙 |
|------|------|--------|
| PDF | 精确排版、压印 | Free（水印）/ Pro（无水印）|
| Word (.docx) | 可再编辑 | Pro |
| JSON / YAML | 机器可读、导入导出 | Free |
| HTML | 网页版简历 | Free |
| LaTeX Source | 学术圈偏好 | Pro |
| 图片（PNG / JPG）| 社交媒体分享 | Free |
| Markdown | 程序员友好 | Free |

#### 2.4.2 在线简历（Share Page）
- 唯一永久链接（可以自定义 slug）
- 暗色模式、响应式（移动端自适应）
- 访问统计（谁在什么时候看过、看了多久）
- 密码保护 / 限时链接
- SEO 优化（Google / Baidu 可索引）

#### 2.4.3 第三方集成导出
- 一键投递 BOSS 直聘 / 猎聘 / LinkedIn／智联 / 51job
- 生成 LinkedIn 兼容格式导入文件
- 自动填充招聘平台的在线简历表单（浏览器插件）

### 2.5 投递追踪系统

#### 2.5.1 投递看板
- 状态管道：草稿 → 已投递 → 简历被查看 → 不回 -> 初面 → 二面 → 终面 → Offer → 入职 / 拒绝
- 每个投递记录：公司、岗位、投递时间、投递渠道、笔记
- 看板视图 + 列表视图切换

#### 2.5.2 提醒与跟进
- 投递后 X 天未回应 → 提醒跟进
- 面试倒计时日历
- 批量管理：撤回 / 标记过期 / 批量导出

#### 2.5.3 数据看板
- 投递总量、面试转化率、渠道统计
- 每月投递热力图
- 面试趋势（行业 / 岗位 / 城市）

### 2.6 企业端（招聘方）

#### 2.6.1 职位发布
- 自定义 JD 模板
- 发布至平台内的简历投递页
- 外部渠道同步（BOSS / 猎聘 / LinkedIn / 牛客）

#### 2.6.2 简历筛选
- 简历解析入库（PDF / Word → 结构化字段）
- 标签系统 & 搜索 & 过滤器
- AI 匹配度排序（JD ↔ 简历）
- 批量操作（标记 / 移动 / 回复 / 归档）

#### 2.6.3 沟通管道
- 站内信 / 面试邀约 / 批注反馈
- 面试评分表（可自定义打分维度）
- Offer 管理（模板、发送、接受率统计）

### 2.7 社区与分享

#### 2.7.1 模板市场
- 设计师上传付费模板（分成模式）
- 模板评分、评论、使用统计
- 精选/热门/新上架 榜单

#### 2.7.2 求职经验内容
- 简历写法教程（专栏文章）
- 面经分享（关联岗位）
- 薪资讨论（匿名模式）

#### 2.7.3 简历点评
- 用户公开简历 → 社区匿名点评
- 专家付费点评（平台审核专家资历）

## 三、非功能需求

### 3.1 性能指标
- 编辑器输入延迟 < 100ms（典型操作）
- PDF 导出 < 5s（中文字体渲染优先）
- 首屏加载 < 2s（SSR + CDN）
- API P99 响应 < 500ms
- 并发支持：1000+ 用户同时编辑

### 3.2 可用性
- SLA 99.9% 以上（核心服务）
- 灾难恢复 RTO < 30min, RPO < 5min
- 计划内停机窗口发布（夜间维护）

### 3.3 安全性
- 全站 HTTPS
- 用户数据加密存储（AES-256 静态加密）
- 简历内容沙箱（防止 XSS/脚本注入）
- 导出文件内容污染水印（追踪泄露源头）
- 敏感信息掩码（电话、邮箱在公开共享页可隐藏）
- GDPR / PIPL（个人信息保护法）合规
- 防爬虫（验证码、速率限制、IP 信誉评分）

### 3.4 可扩展性
- 微服务架构（编辑器服务、AI 服务、导出服务、用户服务可独立伸缩）
- 插件化模板引擎（新增模板不重启）
- 国际化和本地化（i18n 中/英/日/德/西多语言 UI）

### 3.5 兼容性
- 浏览器：Chrome / Firefox / Safari / Edge（近 2 个主要版本）
- 移动端：iOS Safari / 安卓微信内置浏览器
- PDF/A 长期归档格式支持

## 四、技术架构

### 4.1 前端
- **渲染层：** Next.js (React) + SSR / SSG（内容页 SEO）
- **编辑器内核：** TipTap + ProseMirror
- **富媒体排版：** React-PDF / Puppeteer 无头渲染导出
- **状态管理：** Zustand / TanStack Query
- **样式：** Tailwind CSS + Radix UI 组件库
- **移动端：** React Native (iOS + Android) 或 Flutter
- **PWA 支持：** 离线编辑 + Service Worker 缓存

### 4.2 后端
- **API 层：** Node.js (NestJS / Fastify) + GraphQL / REST
- **语言备选：** Go (高性能导出服务、PDF 引擎)
- **数据库：**
  - PostgreSQL（用户、简历、投递记录 — 关系数据）
  - MongoDB（简历快照、版本历史 — 文档型）
  - Elasticsearch（全文搜索、简历 / JD 全文索引）
- **缓存：** Redis（会话、模板缓存、导出结果缓存）
- **队列：** Bull / RabbitMQ（PDF 导出、AI 任务）
- **对象存储：** AWS S3 / MinIO（头像、导出文件、模板素材）

### 4.3 AI 层
- LLM 调用（OpenAI / Anthropic / 通义千问 / DeepSeek API）
- 提示词工程服务：JD 解析、润色、翻译、评分 独立模板
- 嵌入向量索引：简历 ↔ JD 相似度匹配（企业端）
- RAG 管道：基于用户行业经验库的润色建议
- ATS 模拟器：基于规则引擎 + ML 分类器

### 4.4 DevOps & 基础设施
- **容器化：** Docker + Kubernetes (EKS / GKE / 阿里云 ACK)
- **CI/CD：** GitHub Actions / GitLab CI + ArgoCD
- **观测：** Prometheus + Grafana + Sentry + OpenTelemetry
- **日志：** ELK Stack / Loki
- **CDN：** Cloudflare / AWS CloudFront（模板资源、导出文件）
- **DDoS 防护：** Cloudflare WAF 或 Aliyun WAF

### 4.5 第三方服务集成清单
| 服务 | 用途 |
|------|------|
| SendGrid / Amazon SES / 阿里云邮件 | 发送注册验证、投递提醒 |
| Twilio / 阿里云短信 | 短信验证 |
| Stripe / 支付宝 / 微信支付 | 支付 |
| Auth0 / 微信登录 / LinkedIn OAuth | 第三方登录 |
| OpenAI / DeepSeek / Anthropic | AI 功能 |
| Puppeteer / Playwright | PDF 导出 |
| GitHub / GitLab | OAuth 登录（技术用户）|
| Google Drive / Dropbox | 云导入 / 云备份 |

### 4.6 数据流架构
```
[Client (Web/Mobile)] → CDN → WAF → Load Balancer
                                           ↓
[Edge API Gateway] → Rate Limiter → Auth Service
                        ↓
            ┌───────────┬──────────────┬──────────────┐
            ↓           ↓              ↓              ↓
      Postgres      Redis       AI Service      Queue Worker
  (用户/简历/投递)  (缓存/会话)   (润色/评分/JD匹配)  (导出/邮件)
            ↓                       ↓              ↓
      Elasticsearch          LLM API pool     S3 / MinIO
     (全文搜索/匹配)         (OpenAI/通义等)   (文件存储)
```

## 七、法律与合规

### 7.1 隐私与数据保护
- 用户同意采集哪些字段、保留时长、删除流程
- 简历被企业检索前的用户主动同意（opt-in 模式）
- 禁止爬虫抓取公开简历页内容（robots.txt + 法律条款）

### 7.2 版权
- 模板版权归设计师 / 平台所有
- 用户创作内容版权归用户（平台不可商用用户数据）
- AI 生成内容的版权归属声明

### 7.3 合规
- 全球：GDPR（EU 用户数据跨境存储）
- 中国：PIPL《个人信息保护法》、网络安全等级保护
- 招聘方使用条款（禁止歧视性筛选）

## 九、风险管理

| 风险 | 可能性 | 影响 | 缓解措施 |
|------|--------|------|----------|
| PDF 排版偏差 | 高 | 中 | 多引擎对比测试，PDF/A 校验，用户反馈反馈报告 |

| 招聘平台 API 不开放 | 中 | 中 | 优先走浏览器插件 / 手动导出导入兜底 |

## 十、开放性问题与待决策事项

- [ ] 免费层限制用户创建几份简历？3 份 or 5 份？
