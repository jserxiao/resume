# 程序化装饰图形创建指南

项目架构天然支持通过纯前端代码程序化创建自定义装饰图形，无需接入任何外部 API，无需改造任何现有代码。

---

## 数据流闭环

```
程序化代码生成 DecorationAnchor[]（0-100 百分比坐标）
        ↓
组装为 CustomDecorationDefinition
        ↓
调用 store.saveCustomDecoration(decoration)   ← 已有 API
        ↓
左侧面板自动显示、可拖拽放置
        ↓
addBlockFromCustomDecoration → 创建块实例   ← 已有 API
        ↓
DecorationSvgRenderer 渲染                    ← 已有组件
```

---

## 核心类型

程序化创建只需要填写以下三个类型：

### 1. 锚点 `DecorationAnchor`

```typescript
interface DecorationAnchor {
  x: number;           // 0-100 百分比坐标
  y: number;           // 0-100 百分比坐标
  handleOut?: { x: number; y: number } | null;  // 贝塞尔出控制柄（影响当前点到下一个点之间的弧度）
  handleIn?:  { x: number; y: number } | null;  // 贝塞尔入控制柄（影响上一个点到当前点之间的弧度）
}
```

- 坐标系为 **0-100 百分比**，与参数化图形公式天然契合
- `handleOut` / `handleIn` 为可选，不填则为直线段，填写则为二次贝塞尔曲线
- 控制柄的计算规则：优先使用 `handleOut`，若不存在则使用 `handleIn`，两者都没有则直线连接

### 2. 路径 `DecorationPath`

```typescript
interface DecorationPath {
  id: string;                    // 随机 ID，如 uuid().slice(0, 8)
  anchors: DecorationAnchor[];   // 锚点列表
  isClosed: boolean;             // 是否闭合（闭合路径首尾相连）
  fillColor: string;             // 填充色，如 '#1a56db'，'transparent' 表示无填充
  strokeColor: string;           // 描边色
  strokeWidth: number;           // 描边宽度（0-100 百分比）
  clipRect?: {                   // 可选裁剪矩形
    x: number; y: number;
    width: number; height: number;
  } | null;
  edgeColors?: string[];         // 可选逐边着色
}
```

- `strokeWidth` 也是 0-100 百分比值，一般取 `1` ~ `3` 即可
- 一个装饰可以包含**多条路径**，每条路径独立设置颜色和样式

### 3. 装饰定义 `CustomDecorationDefinition`

```typescript
interface CustomDecorationDefinition {
  id: string;              // 唯一 ID，格式 'cde-xxxxxxxx'
  name: string;            // 显示名称
  paths: DecorationPath[]; // 路径列表
  stageWidth?: number;     // 像素，决定放置时块的默认宽度
  stageHeight?: number;    // 像素，决定放置时块的默认高度
  createdAt: number;       // Date.now()
  updatedAt: number;       // Date.now()
}
```

## 示例：程序化生成一朵 5 瓣花

```typescript
import { useResumeStore } from '@/store';
import { v4 as uuid } from 'uuid';
import type { CustomDecorationDefinition, DecorationPath, DecorationAnchor } from '@/types';

function generateFlower(petals: number, innerRatio: number): CustomDecorationDefinition {
  const anchors: DecorationAnchor[] = [];
  const n = petals * 2; // 每瓣2个点（外尖 + 内凹）

  for (let i = 0; i < n; i++) {
    const angle = (i / n) * Math.PI * 2 - Math.PI / 2;
    const isOuter = i % 2 === 0;
    const r = isOuter ? 48 : 48 * innerRatio;
    anchors.push({
      x: 50 + Math.cos(angle) * r,
      y: 50 + Math.sin(angle) * r,
    });
  }

  const path: DecorationPath = {
    id: uuid().slice(0, 8),
    anchors,
    isClosed: true,
    fillColor: '#1a56db',
    strokeColor: '#1a56db',
    strokeWidth: 1.5,
  };

  return {
    id: `cde-${uuid().slice(0, 8)}`,
    name: `${petals}瓣花`,
    paths: [path],
    stageWidth: 120,
    stageHeight: 120,
    createdAt: Date.now(),
    updatedAt: Date.now(),
  };
}

// 保存到 store，立即出现在左侧面板
const store = useResumeStore.getState();
store.saveCustomDecoration(generateFlower(5, 0.4));
```

## 更多程序化图形公式参考

### 正弦波

```typescript
function generateWave(cycles: number, amplitude: number): DecorationAnchor[] {
  const steps = 40;
  const anchors: DecorationAnchor[] = [];
  for (let i = 0; i <= steps; i++) {
    const t = i / steps;
    anchors.push({
      x: t * 100,
      y: 50 + Math.sin(t * Math.PI * 2 * cycles) * amplitude,
    });
  }
  return anchors;
}
```

### 极坐标玫瑰线

```typescript
// r(θ) = cos(kθ)，k 为花瓣参数
function generateRose(k: number, steps: number = 80): DecorationAnchor[] {
  const anchors: DecorationAnchor[] = [];
  for (let i = 0; i <= steps; i++) {
    const theta = (i / steps) * Math.PI * 2;
    const r = Math.cos(k * theta) * 48; // 半径 48，中心 50
    anchors.push({
      x: 50 + r * Math.cos(theta),
      y: 50 + r * Math.sin(theta),
    });
  }
  return anchors;
}
```

### 螺旋线

```typescript
function generateSpiral(turns: number, steps: number = 60): DecorationAnchor[] {
  const anchors: DecorationAnchor[] = [];
  for (let i = 0; i <= steps; i++) {
    const t = i / steps;
    const angle = t * Math.PI * 2 * turns;
    const r = t * 45; // 从中心向外扩展
    anchors.push({
      x: 50 + r * Math.cos(angle),
      y: 50 + r * Math.sin(angle),
    });
  }
  return anchors;
}
```

### 带贝塞尔控制柄的圆角多边形

```typescript
function generateRoundedPolygon(sides: number, cornerRadius: number = 0.15): DecorationAnchor[] {
  const anchors: DecorationAnchor[] = [];
  for (let i = 0; i < sides; i++) {
    const angle = -Math.PI / 2 + (i * 2 * Math.PI) / sides;
    const x = 50 + Math.cos(angle) * 45;
    const y = 50 + Math.sin(angle) * 45;

    // 计算控制柄方向（沿边方向）
    const nextAngle = -Math.PI / 2 + ((i + 1) * 2 * Math.PI) / sides;
    const prevAngle = -Math.PI / 2 + ((i - 1 + sides) * 2 * Math.PI) / sides;
    const handleLen = cornerRadius * 45;

    anchors.push({
      x, y,
      handleOut: {
        x: x + Math.cos(nextAngle - Math.PI / 2) * handleLen,
        y: y + Math.sin(nextAngle - Math.PI / 2) * handleLen,
      },
      handleIn: {
        x: x + Math.cos(prevAngle + Math.PI / 2) * handleLen,
        y: y + Math.sin(prevAngle + Math.PI / 2) * handleLen,
      },
    });
  }
  return anchors;
}
```

### 超椭圆（Squircle）

```typescript
// |x/a|^n + |y/b|^n = 1
function generateSuperEllipse(n: number, steps: number = 60): DecorationAnchor[] {
  const anchors: DecorationAnchor[] = [];
  for (let i = 0; i < steps; i++) {
    const t = (i / steps) * Math.PI * 2;
    const cosT = Math.cos(t);
    const sinT = Math.sin(t);
    const x = Math.sign(cosT) * Math.pow(Math.abs(cosT), 2 / n) * 48;
    const y = Math.sign(sinT) * Math.pow(Math.abs(sinT), 2 / n) * 48;
    anchors.push({ x: 50 + x, y: 50 + y });
  }
  return anchors;
}
```

### 星芒（可调内径比的多角星）

```typescript
function generateBurstStar(points: number, innerRatio: number = 0.4): DecorationAnchor[] {
  const anchors: DecorationAnchor[] = [];
  const total = points * 2;
  for (let i = 0; i < total; i++) {
    const angle = -Math.PI / 2 + (i * Math.PI) / points;
    const isOuter = i % 2 === 0;
    const r = isOuter ? 48 : 48 * innerRatio;
    anchors.push({
      x: 50 + Math.cos(angle) * r,
      y: 50 + Math.sin(angle) * r,
    });
  }
  return anchors;
}
```

## 多路径装饰示例：带边框的花

一个装饰可以包含多条路径，分别设置不同颜色和样式：

```typescript
function generateFramedFlower(): CustomDecorationDefinition {
  // 路径1：花朵
  const flowerAnchors = generateBurstStar(5, 0.4);
  // 路径2：外框圆
  const circleAnchors: DecorationAnchor[] = [];
  for (let i = 0; i < 8; i++) {
    const angle = (i * Math.PI) / 4;
    circleAnchors.push({
      x: 50 + Math.cos(angle) * 48,
      y: 50 + Math.sin(angle) * 48,
      handleOut: { x: 50 + Math.cos(angle + Math.PI / 8) * 62, y: 50 + Math.sin(angle + Math.PI / 8) * 62 },
    });
  }
  for (let i = 0; i < 8; i++) {
    const prevIdx = (i - 1 + 8) % 8;
    const prevMidAngle = ((prevIdx) * Math.PI) / 4 + Math.PI / 8;
    circleAnchors[i].handleIn = {
      x: 50 + Math.cos(prevMidAngle) * 62,
      y: 50 + Math.sin(prevMidAngle) * 62,
    };
  }

  return {
    id: `cde-${uuid().slice(0, 8)}`,
    name: '边框花',
    paths: [
      {
        id: uuid().slice(0, 8),
        anchors: flowerAnchors,
        isClosed: true,
        fillColor: '#1a56db',
        strokeColor: 'transparent',
        strokeWidth: 1,
      },
      {
        id: uuid().slice(0, 8),
        anchors: circleAnchors,
        isClosed: true,
        fillColor: 'transparent',
        strokeColor: '#1a56db',
        strokeWidth: 2,
      },
    ],
    stageWidth: 120,
    stageHeight: 120,
    createdAt: Date.now(),
    updatedAt: Date.now(),
  };
}
```

## 已有几何工具参考

`src/utils/geometry.ts` 中已包含以下参数化形状生成器，可直接复用或参考：

| 函数 | 说明 |
|------|------|
| `generateRectangle(x, y, w, h)` | 矩形 |
| `generateRoundedRect(x, y, w, h)` | 圆角矩形（8锚点+控制柄） |
| `generateCircle(cx, cy, w, h)` | 圆形（8锚点二次贝塞尔近似） |
| `generateEllipse(x, y, w, h)` | 椭圆 |
| `generateTriangle(x, y, w, h)` | 三角形 |
| `generateDiamond(x, y, w, h)` | 菱形 |
| `generateStar(x, y, w, h)` | 五角星 |
| `generateHeart(x, y, w, h)` | 心形（经典参数方程 + 16段二次贝塞尔拟合） |
| `generateArrowRight(x, y, w, h)` | 右箭头 |
| `generateHexagon(x, y, w, h)` | 六边形 |

## 坐标转换备忘

装饰编辑器内部使用**像素坐标**，而 `CustomDecorationDefinition` 存储的是 **0-100 百分比**。程序化生成时：

- 如果直接生成 0-100 百分比坐标，无需转换
- 如果复用 `geometry.ts` 的像素坐标生成器，需要归一化：

```typescript
function pixelToPercent(px: number, py: number, stageW: number, stageH: number, cropX: number, cropY: number, cropW: number, cropH: number): { x: number; y: number } {
  return {
    x: ((px - cropX) / cropW) * 100,
    y: ((py - cropY) / cropH) * 100,
  };
}
```
